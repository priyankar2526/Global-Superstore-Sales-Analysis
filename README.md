# Global-Superstore-Sales-Analysis
📊 Sales Dashboard Analysis

📌 Overview
This project is a Power BI dashboard that analyzes sales, profit, and business performance using a retail dataset.

📊 Features
- Total Sales & Profit Analysis
- Sales by Category & Region
- Segment-wise Insights
- Shipping Performance
- Interactive Filters

🛠 Tools
- Power BI
- Excel
- DAX

📈 Insights
- Technology has highest sales
- Consumer segment leads
- Western region performs best
